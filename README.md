## ğŸ¨ Key Components

### Context Providers
- **CartContext**: 
  - Add/remove products
  - Update quantities
  - Calculate totals
  - Persist to localStorage
  
- **WishlistContext**: 
  - Add/remove products
  - Toggle wishlist status
  - Check if product is wishlisted
  - Persist to localStorage

### Main Pages
- **Home**: Landing page with hero section and featured products
- **Shop**: Product catalog with advanced filtering and search
- **ShopDetail**: Individual product details with add to cart/wishlist
- **Cart**: Shopping cart drawer (global component)
- **Wishlist**: Dedicated page for saved products
- **About**: Company information and mission
- **Contact**: Contact form with Google Maps and social links

### Shopping Cart System

```javascript
// Available cart methods

cart.removeFromCart(productId)
cart.updateQuantity(productId, newQuantity)
cart.clearCart()
cart.getCartTotal()
cart.getCartCount()
```

### Wishlist Management

```javascript
// Available wishlist methods
wishlist.addToWishlist(product)
wishlist.removeFromWishlist(productId)
wishlist.toggleWishlist(product)
wishlist.isInWishlist(productId)
wishlist.clearWishlist()
```

### Data Persistence

Both cart and wishlist use localStorage:
- **Cart**: Saved as `medifitCart`
- **Wishlist**: Saved as `medifitWishlist`
- Automatically loads on app initialization
- Survives page refresh and browser restart
